# Study

자격증 학습과 실무 역량 강화를 함께 정리하는 페이지입니다.

## Main Tracks

### Backend / System Design

- Java
- Spring Boot
- REST API
- 데이터베이스 설계
- 운영 환경 구성

### Infrastructure / Operations

- Linux
- Nginx / Apache / Tomcat
- Docker / Podman
- 배포 자동화
- 장애 대응

### Data Systemization

- 데이터 분석 결과 시스템화
- 분석가와 개발자 간 요구사항 연결
- 분석 결과 검증 및 서비스 반영 기준

### Security / Audit

- 정보보안
- 접근통제
- 로그관리
- 내부통제
- CISA
- ISMS-P

### Project Management

- PMP
- 업무일지
- 일일보고
- 요구사항 정리
- 고객 커뮤니케이션

## Writing Rule

학습 글도 단순 요약이 아니라 실무 적용 기준을 중심으로 작성합니다.

```text
개념 -> 실무 적용 기준 -> 예시 -> 주의점 -> 정리
```
