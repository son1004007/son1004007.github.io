# Study

자격증 학습과 실무 역량 강화를 함께 정리하는 페이지입니다.

이 페이지는 단순 학습 목록이 아니라, 현재 경력 방향과 연결되는 학습 주제를 정리하기 위한 공간입니다.

## 학습 방향

현재 학습은 아래 목적을 기준으로 정리합니다.

1. 백엔드 개발 역량 강화
2. Linux / 웹서버 / 컨테이너 기반 운영 역량 강화
3. 데이터 분석 결과를 시스템화하는 역량 강화
4. 보안/감사/내부통제 직무 전환 가능성 확보
5. PMP, CISA, ISMS-P 등 자격증 학습을 실무 언어로 변환

## Main Tracks

### Backend / System Design

- Java
- Spring Boot
- REST API
- 데이터베이스 설계
- 인증/권한
- 운영 환경 구성

### Infrastructure / Operations

- Linux
- Rocky Linux / Ubuntu
- Nginx / Apache / Tomcat
- Docker / Podman
- 배포 자동화
- 장애 대응

### Database

- PostgreSQL
- Oracle
- Tibero
- SQL 튜닝
- 데이터 모델링
- 분석 결과 테이블 설계

### Data Systemization

- 데이터 분석 결과 시스템화
- 분석가와 개발자 간 요구사항 연결
- 분석 결과 검증 및 서비스 반영 기준
- 배치, API, 화면 간 역할 분리

### Security / Audit

- 정보보안
- 접근통제
- 로그관리
- 내부통제
- CISA
- ISMS-P
- IT 감사/감리 관점의 기술 정리

### Project Management

- PMP
- 업무일지
- 일일보고
- 요구사항 정리
- 고객 커뮤니케이션
- 분석 범위와 개발 범위 조율

## 작성 기준

학습 글도 단순 요약이 아니라 실무 적용 기준을 중심으로 작성합니다.

```text
개념 -> 실무 적용 기준 -> 예시 -> 주의점 -> 정리
```

## 우선 정리할 학습 글

- PMP 학습 내용을 실무 프로젝트 관리 언어로 정리하는 방법
- CISA 학습 내용을 IT 내부통제 관점으로 정리하는 방법
- ISMS-P 학습을 웹 서비스 운영 기준으로 연결하는 방법
- 데이터베이스 설계 이론을 분석 결과 시스템화에 적용하는 방법
- Linux 장애 대응 절차를 운영 체크리스트로 만드는 방법
